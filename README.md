
# Payment App

## Run Locally

Clone the project

```bash
  git clone https://github.com/Muhammad-Kamran-Dev/gpt-3.git
```

Go to the project directory

```bash
  cd project_modern_payment_app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
