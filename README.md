## Run Locally

Clone the project

```bash
  git clone https://github.com/Muhammad-Kamran-Dev/project_modern_payment_app.git
```

Go to the project directory

```bash
  cd project_modern_payment_app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
